# jswank/oracle-jdk7

Oracle JDK 7

To build:

Copy the sources down and do the build-

```
$ docker build --rm --tag=jswank/oracle-jdk7 .
```

To run:

```
$ docker run -t -i jswank/oracle-jdk7
```
